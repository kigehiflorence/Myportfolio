## Deployment

### Deploy to Vercel

1. **Fork this repository** to your GitHub account
2. **Sign up/Login to Vercel** at [vercel.com](https://vercel.com)
3. **Import your repository** from GitHub
4. **Deploy** - Vercel will automatically detect the project and deploy it

#### Manual Deployment Steps:
1. Install Vercel CLI: `npm i -g vercel`
2. Login to Vercel: `vercel login`
3. Deploy: `vercel --prod`

### Local Development
```bash
# Clone the repository
git clone [https://github.com/kigehiflorence/Terminal-Portfolio.git](https://github.com/kigehiflorence/Terminal-Portfolio.git)

# Navigate to project directory
cd Terminal-Portfolio

# Start local server (Python)
python -m http.server 8000

# Or use Node.js
npx http-server

# Open browser to http://localhost:8000
